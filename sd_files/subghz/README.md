# CC1101 SubGHz Signals Folder

# Place captured .sub or .txt files here

This folder stores RF signals captured by the CC1101 module.

## Supported Formats

- `.sub` - Flipper Zero SubGhz format
- `.txt` - RAW signal format

## Example Flipper .sub file

```
Filetype: Flipper SubGhz Key File
Version: 1
Frequency: 433920000
Preset: FuriHalSubGhzPresetOok650Async
Protocol: Princeton
Bit: 24
Key: 00 00 00 00 00 12 34 56
```

## Protocols Supported

- Princeton (24-bit)
- CAME (12-bit)
- Nice FLO (12-bit)
- Linear (10-bit)
- Holtek (12-bit)
- PT2262 (24-bit)
