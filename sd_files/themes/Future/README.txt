Tema Future - Instruções

Este tema está configurado com cores Neon (Cyan/Magenta) e efeitos de LED.
Para adicionar ícones e imagens de fundo, coloque os arquivos JPG/PNG nesta pasta e adicione as referências no arquivo theme.json.

Exemplo de adição no theme.json:
{
  ...
  "wifi": "wifi_icon.jpg",
  "ble": "ble_icon.jpg",
  "boot_img": "boot_logo.jpg"
}

Imagens suportadas:
- wifi, ble, ethernet, rf, rfid, fm, ir
- Arquivos (gerenciador de arquivos)
- gps, nrf, interpreter, clock
- Outros, usb, Conectar, config
- boot_img (Tela de boot)
- boot_sound (Arquivo .wav para som de boot)
